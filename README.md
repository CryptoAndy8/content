# content

Purpose
- Швидко збирати TG пости однаковим стилем.

Rules
- Тільки короткі дефіси (-).
- Короткі тези, без води, 3-6 булетів.
- Лінки чисті, без utm.

How to use
- `python src/tg_post_formatter.py` для прикладу.
- Або імпортуй `render_post` у свій скрипт.

Folders
- `/docs` шаблони
- `/src` утиліти
- `/assets` картинки
